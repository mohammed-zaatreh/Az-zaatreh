# Mohammed Asha Portfolio

A cinematic software engineering portfolio built to showcase backend engineering experience, systems thinking, and original software projects.

The website is deployed through Cloudflare and designed as a fully static experience powered by custom animations, Three.js particle systems, and modern frontend techniques.

## Overview

This portfolio was created to present:

* Backend engineering experience
* Production software development practices
* Original research-driven projects
* Professional experience and technical capabilities
* International software engineering opportunities

The design focuses on creating an immersive experience rather than a traditional portfolio layout, combining engineering storytelling with interactive visual systems.

## Featured Projects

### MOIRA

Infrastructure Intelligence Engine

MOIRA models infrastructure as a network of connected nodes and evaluates how value, dependency, and influence move through a system.

Technologies:

* Spring Boot
* PostgreSQL
* Graph Analysis
* Risk Modeling

### SERAPH

Semantic Pattern Intelligence

SERAPH explores conceptual similarity beyond keywords by comparing information through meaning and relationships rather than exact text matching.

Technologies:

* Spring Boot
* PostgreSQL
* Semantic Analysis
* Retrieval Systems

## Local Development

Clone the repository:

```bash
git clone https://github.com/your-username/your-repository.git
```

Enter the project directory:

```bash
cd your-repository
```

Open the project:

```bash
open index.html
```

Or run a local server:

```bash
python -m http.server 8000
```

Then visit:

```text
http://localhost:8000
```
